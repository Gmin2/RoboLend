# RoboLend

Lend and borrow against tokenized stocks on Robinhood Chain. Deposit TSLA, AMZN, PLTR, NFLX, AMD as collateral and borrow WETH.

## Getting Started

```bash
# Frontend
cd apps
pnpm install
pnpm dev
```

```bash
# Contracts
forge build
forge test
```
